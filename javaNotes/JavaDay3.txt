Object :-   It is an instance of the class. Every object is a isolated unit. i.e once any content stored, that cannot 		be travels to other object until we specify.

		<Class_Name> <object> = new <Class_Name>([arg]);

	Once object is created, we can access class members(variables/methods) as follows
			object.variable/method_name

--------------------------------------
Access Specifiers :- these are used to specify how to access class members either inside / outside the class. 

private :-  it allows to access class members only inside the class. 
public  :-  it allows to access class memebrs inside and outside the class. 
protected :-  it allows to access one class members into another specific class only. 

Note :-  by default every class member is a public. 
---------------------------------------------------------
Classification of Class Members(Variables/methods)
		there are two types of class members
			1.  Non-Static Members / Instance Members
			2.  Static Members

A Non-Static Members are used to whenever required multiple of copies data. All Instances/Objects comes under 
Non-Static process. 

Static Members :-  All members are called as static members when those members are precdede with "static" keyword.
			Static Variables
			Static Methods
1.  Static members are not occupies any space in Object. It has separate memory called class memory. 
2.  A Static method can handle only static variables whereas Non-static methods can handle both static and non-static 		variables.
3.  If a class having all static members, then no need to create an object for that class, call the members with class name only. 

-----------------------------------
Constructor:- It is a special method of the class which has the same name of class. 
purpose of the constructor is to specify initial process for the object. i.e. initialize instance variables at the time of object creation. 

1. Construtor will be executed whenever object is created for the class. 
2. It can take parameters, for those parameters, we have to pass values at the time of object creation.
3. A class can have multiple constructors, here each constructor should have different no of parameters.
